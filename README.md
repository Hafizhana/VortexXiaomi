# **Info VortexXiaomi**
**VortexXiaomi** is a Magisk Module designed and created especially for game performance. Vortex optimizes CPU, GPU, Internet, and System Scheduling to enhance game experience. Vortex is lightweight, auto-applies during games, and is considered safe to use for Mediatek Users.

# **Supported Root**
 •[Magisk](https://github.com/topjohnwu/Magisk)

Instruction:

• Download and flash the zip in Magisk.

• **If you have old version of the module, delete first the old module.**

• Reboot the device.

• Enjoy.
